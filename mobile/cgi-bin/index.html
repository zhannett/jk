Welcome!

My name is Janet. I am a Web Developer.

My interest to mobile technologies pushed me to take "Foundations of Mobile Business Technologies" course at the Unoversity of Toronto.

This is my first mobile website created as a final project for this course. The main goal of this site is demonstrating 20 mobile web features that I have learned and getting my next contract as a Mobile Web Developer.

20 mobile web features learned during the course

  1. Conditional media Queries (for every page)
  2. Using ordered lists with access keys (Home page + navigation on every page) - for following browsers ...
  3. Using text overflow with elipsis
  4. Use of pseudo-classes - for every link on the website excluding navigation links - for following browsers ...
  5. Using border image
  6. Use of HTML5 input types
  7. Using geolocation and error handling
  8. Use of appropriate map
  9. Using sprites
  10. Using reset.css
